# Student-Marks-Analysis-Dashboard
Project Title / Headline

Student Marks Analysis Dashboard: Academic Performance Insights Dashboard

A dynamic and interactive dashboard developed to analyze student academic performance, identify subject-wise trends, and provide actionable insights for educational improvement.

2. Short Description / Purpose

The Student Marks Analysis Dashboard is an analytical and visually engaging dashboard created using Microsoft Excel to evaluate student performance across Mathematics, Reading, and Writing subjects. The dashboard helps educators and administrators monitor academic progress, identify high and low performers, compare subject-wise scores, and make data-driven decisions to improve learning outcomes.

3. Tech Stack

The dashboard was built using the following tools and technologies:

Microsoft Excel – Primary platform used for data analysis and dashboard creation.
Pivot Tables – Used for data summarization and aggregation.
Pivot Charts – Used to create interactive visualizations.
Slicers and Filters – Enabled dynamic filtering and user interaction.
Conditional Formatting – Highlighted performance trends and key insights.
Excel Formulas and Functions – Used for calculations such as averages, percentages, and KPIs.
Data Cleaning Techniques – Removed duplicates, handled missing values, and prepared the dataset for analysis.
4. Key Features
Analysis of student marks in Mathematics, Reading, and Writing.
Overall score percentage calculation and performance evaluation.
Subject-wise comparison of student performance.
Gender-wise academic performance analysis.
Performance analysis based on parental education levels.
Test preparation course impact analysis.
Identification of top-performing and low-performing students.
Interactive dashboard with filters and slicers for detailed exploration.
5. Business Problem / Objective

Educational institutions often collect large amounts of student performance data but face challenges in extracting meaningful insights. This dashboard aims to transform raw student marks data into a clear and interactive visual format, enabling educators to monitor academic performance, identify learning gaps, and support informed decision-making for improved educational outcomes.

6. Dashboard Insights
Students who completed test preparation courses generally achieved higher scores.
Reading and Writing scores show a strong positive correlation.
Performance varies across different parental education levels.
Female students tend to perform better in Reading and Writing, while Mathematics performance is relatively balanced.
The dashboard helps identify students requiring additional academic support.
